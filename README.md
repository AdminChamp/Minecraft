# Minecraft
There are some Minecraft maps, hacks, mods, textures and other
To apply these maps:
1. Download the map
2. Extract the folder
3. Press Win + R
4. Type %appdata%
5. Go to .minecraft\saves
6. Move the maps folder to this directory
7. Launch the Minecraft Launcher
8. Have fun!
To apply these textures:
1. Download the texture
2. Extract the folder
3. Press Win + R
4. Type %appdata%
5. Go to .minecraft\resourcepack
6. Move the texture folder to this directory
7. Launch the Minecraft Launcher
8. Have fun!
